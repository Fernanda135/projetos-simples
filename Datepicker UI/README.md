# Datepicker UI

Interface de seleção de datas desenvolvida para prática de componentes visuais.

## Descrição

O projeto apresenta um layout de datepicker, simulando um componente comum em aplicações web.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo do Projeto

Praticar a criação de componentes interativos e manipulação de eventos.
