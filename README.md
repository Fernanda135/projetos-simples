# Projetos Simples

Este repositório reúne **projetos simples e mini sistemas** desenvolvidos com o objetivo de praticar programação e aplicar conceitos aprendidos.

Os projetos aqui presentes são voltados para aprendizado, organização de código e construção de portfólio acadêmico.

Cada pasta contém seu próprio código-fonte e, quando necessário, instruções para execução ou uso do projeto.

##  Objetivo do Repositório

- Praticar conceitos fundamentais de programação  
- Aplicar conhecimentos adquiridos em sala de aula  
- Organizar projetos simples de forma clara e estruturada  
- Servir como base para evolução técnica e profissional
