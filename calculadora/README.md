# Calculadora

Calculadora simples desenvolvida para operações matemáticas básicas.

## Descrição

Permite realizar cálculos como soma, subtração, multiplicação e divisão por meio de uma interface simples.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo do Projeto

Praticar lógica de programação e manipulação de eventos.
