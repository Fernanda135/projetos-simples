# Basic HTML Website

Site simples desenvolvido utilizando apenas HTML, com o objetivo de praticar a estrutura básica de páginas web.

## Descrição

O projeto explora elementos fundamentais do HTML, como cabeçalhos, parágrafos, links e organização de conteúdo.

## Tecnologias Utilizadas

- HTML5

## Objetivo do Projeto

Compreender a estrutura de um site básico e reforçar conceitos iniciais de desenvolvimento web.
