# Accessible Form UI

Projeto de interface de formulário acessível, desenvolvido com foco em boas práticas de usabilidade e acessibilidade na web.

## Descrição

Este projeto apresenta um formulário com estrutura semântica adequada, visando melhorar a experiência de usuários, incluindo pessoas com necessidades de acessibilidade.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Objetivo do Projeto

Praticar a criação de interfaces acessíveis e compreender a importância da semântica e da usabilidade no desenvolvimento web.
