# CV

Projeto de currículo digital desenvolvido em formato web.

## Descrição

Apresenta informações profissionais e acadêmicas organizadas em uma página simples, com foco em clareza e apresentação visual.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Objetivo do Projeto

Praticar a criação de páginas informativas e organização de conteúdo para apresentação profissional.
